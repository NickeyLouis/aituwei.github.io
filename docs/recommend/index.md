## 🤖 软件 

### [yarr](https://github.com/nkanaev/yarr)

基于 `Web` 的 `RSS` 阅读器，可使用 `Docker` 快速体验：

```shell
mkdir -p ./yarr/data
docker run -it -p 7070:7070 -v $(pwd)/yarr/data:/data wbsu2003/yarr
```

总体来说还是挺简洁轻量的：

![yarr](https://images-1252557999.file.myqcloud.com/uPic/yarr.jpg)
